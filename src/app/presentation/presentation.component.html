Développeur Web/Web Mobile

L’évolution rapide du monde du web ainsi que l’ascendance des nouvelles technologies font du développeur un métier incontournable. Les besoins de la clientèle évoluent sans cesse et le développeur ou appelé encore programmeur spécialisé ou technicien informatique doit se conformer aux exigences du secteur d’autant plus que c’est le poste le plus recherché par toutes les entreprises de presque toutes les catégories en ce moment.

Mes objectifs au sein de l'AFPA

Développer la partie front-end d’une application web ou web mobile en intégrant les recommandations de sécurité :

maquettage d'une application - Réalisation d'une interface utilisateur web statique et adaptable - Développement d'une interface utilisateur web dynamique - Réalisation d'une interface utilisateur avec une solution de gestion de contenu ou e-commerce

Développer la partie back-end d’une application web ou web mobile en intégrant les recommandations de sécurité :

Création d'une base de données - Développement des composants d’accès aux données - Développement de la partie back-end d’une application web ou web mobile - Elaboration et mise en œuvre des composants dans une application de gestion de contenu ou e-commerce
